About Retro Gaming Web - Windows 98 Edition

Welcome to our online preservation project dedicated to classic computing. We use advanced v86 virtualization technology to run a full Windows 98 environment directly in your modern web browser using WebAssembly (WASM).

Our library features legendary titles such as Civilization II, Red Alert 2, and StarCraft. By leveraging browser-based emulation, we ensure these historic games remain accessible without the need for complex legacy hardware or local installations.

This site serves as a technical demonstration of high-performance web emulation and as a digital museum for the golden era of PC gaming. All games are configured for optimal performance in the v86 environment.
